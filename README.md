Este codig simula un "groan tube" usando el acelerometro interno de un telefono movil. Al inclinar el telefono, la pelota se mueve y segun la posicion a la que la llevemos controlamos el volumen y la frecuencia.

## EMPEZAR USO

Para que funcione, le tenemos que dar a la flecha de arriba a la izquierda. Una vez empieza a sonar, debajo del recuadro a la derecha, tocamos "Enable motion", esto hara que la pelota empiece a moverse.

# CONTROLES

Debajo del recuadro tenemos dos opciones que nos permiten invertir la direeccion en la que se mueve la pelota.

Si movemos la pelota hacia abajo, bajamos el volumen, si la llevamos arriba lo subimos.

Si movemos la pelota hacia la derecha, bajamos la frecuencia (sonido más grave), si la movemos a la derecha la subimos (sonido más agudo).
